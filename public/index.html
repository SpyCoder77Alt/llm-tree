LLM Family Tree

About

This visualization attempts to map the genesis of large language model (LLM) research from the initial attention paper to the present (early 2025). Because many models are still closed source it is difficult to know exactly which model influenced which, but it is safe to say that all of these models were influenced by the transformer architecture, which is why that is the root of this tree. If you have any feedback or suggestions let me know. This is a work in progress and could change significantly over time.

Data sources: Academic papers, arXiv publications, and official model documentation.
